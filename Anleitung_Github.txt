Vorgehensweise Github Clone
$ cd <dir>  # In das gewünschte Verzeichnis gehen. Dabei wird automatisch im Nachfolgenden der Ordner "ha-as-rcpsp" erstellt. Daher keinen Ordner erstellen!!!
$ git clone https://github.com/rb4k/ha-as-rcpsp.git # Verzeichnis wird kopiert
$ git branch
$ git status


Onlineabruf der Datei
$ git commit -am "<msg>"
$ git pull

Arbeiten in der Datei
$ git add .
$ git commit -am "<msg>"

Wechseln zwischen Branch / Vorgehensweise Github Committing with the Command Line
$ git add .
$ git commit -m "Commit Namen" # Commit wird durchgeführt im aktuellen Branch
$ git push # Zum Branch hinzufügen
